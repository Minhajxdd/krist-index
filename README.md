# 🛒 Krisht eCommerce UI

A simple and responsive eCommerce landing page built using **Tailwind CSS**, **HTML**, and **JavaScript**.  
This project focuses on the frontend layout only — no backend functionality is included.

![Preview](./assets/preview/kirst_preview.png)

## 🚀 Features

- Clean and modern UI
- Fully responsive layout
- Tailwind CSS utility classes
- Built with semantic HTML5
- Includes basic interactivity with vanilla JavaScript

## 📁 Technologies Used

- HTML5
- Tailwind CSS
- JavaScript (optional interactivity)
